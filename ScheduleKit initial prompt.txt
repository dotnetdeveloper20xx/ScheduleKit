## Database configuration

```csharp
// In-memory for development (default)
builder.Services.AddScheduleKit(options =>
{
    options.UseInMemoryDatabase("ScheduleKitDemo");
});

// SQL Server for production
builder.Services.AddScheduleKit(options =>
{
    options.UseSqlServer(builder.Configuration.GetConnectionString("ScheduleKit"));
});
```

The demo app uses an environment variable to switch:

```json
// appsettings.Development.json
{
  "ScheduleKit": {
    "UseInMemory": true
  }
}

// appsettings.Production.json
{
  "ScheduleKit": {
    "UseInMemory": false
  },
  "ConnectionStrings": {
    "ScheduleKit": "Server=...;Database=...;"
  }
}
```
